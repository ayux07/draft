# INDREV — UI & Feature Map

## 🏠 HOME
- **Hero**: Halftone dot bg + diagonal stripe accent. Bangers headline. Yellow accent band.
- **Stats bar**: Yellow bg band. 6+ Curators, 120+ Pieces, 100% Authenticated.
- **Category grid**: Men · Women · Kids · Unisex · Accessories.
- **New Arrivals**: 8 products across all genders.
- **Trust Pillars**: Verified Sellers, Authenticated Pieces, Protected Payments.

## 🔍 BROWSE
- **Search**: Live text search (name / brand).
- **Filters**: Gender tabs, Dynamic Categories, Condition, Max price slider.
- **Grid**: 1→2→3→4 cols, neo-cards.

## 👔 PRODUCT DETAIL
- **Image well**: Flat color bg + halftone overlay. Starburst tag badge.
- **Meta grid**: Condition / Size / Category / Brand in sharp 3px cells.
- **Actions**: Add to Cart (yellow neo-btn), Size Guide, Authenticity Guarantee.

## 🛒 CART DRAWER
- **Slide-in**: White panel, 3px left border.
- **Rows**: Product details, Qty stepper, Remove action.
- **Checkout**: Total calculation + Proceed button.

## 🔐 AUTH (Login/Signup)
- **Modal**: 3px border, 8px hard shadow.
- **Theming**: Comic-style floating labels with yellow highlights.

## 🔒 ADMIN PANEL
- **Dashboard**: Revenue stats, inventory count, order totals.
- **Products**: Full CRUD (Add/Edit/Delete/Hide).
- **Orders**: Status updates (Pending → Shipped → Delivered).
- **Users**: Access control (Ban/Unban).
- **Content**: Homepage featured items management.

## 🛠️ SHELL & DATA
- **Navbar**: Sticky, white bg, Bangers logo, desktop/mobile responsive.
- **Persistence**: LocalStorage sync for Cart, Wishlist, Orders, Auth.
- **Data**: Centralized mock data in `mockData.js`.
